API Documentation

Introduction

Welcome to the API documentation. This document provides detailed information about the API endpoints and how to use them.

Base URL

The base URL for all API endpoints is: http://-your-above-url-

Authentication

All protected routes require a valid JWT token to be included in the Authorization header of the HTTP request. To obtain a JWT token, send a POST request to /v1/auth/login with valid user credentials. The response will include a JWT token that you can use for authentication.
